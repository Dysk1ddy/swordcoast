# CATEGORY 3 — RELIGIOUS / DIVINE LANGUAGE

DnD uses deity-based systems. Aethrune should not.

## Replace

- Shrine → **Sanctum / Lantern Post / Quiet Hall**
- Priest / Acolyte → **Attendant / Keeper**
- Holy / Blessed → **Stabilized / Purified / Treated**

Example:
- Shrine stores → **Sanctum provisions**
- Priest caches → **Keeper reserves**
