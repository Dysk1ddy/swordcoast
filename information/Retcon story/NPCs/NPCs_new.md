# AETHRUNE — NPCs (RETCONNED V1)

## PURPOSE
Fully retconned NPC roster aligned to Aethrune.
- Removes DnD-specific lore
- Aligns all characters to systems, factions, and infrastructure themes
- Preserves gameplay roles and narrative function
- No dialogue included (structure + characterization only)

---

# CORE DESIGN PRINCIPLE

NPCs are not “fantasy archetypes.”

They are:
- system participants
- survivors of infrastructure collapse
- agents of control, truth, or resistance

Each NPC should clearly align with:
- Route Control
- Information Control
- System Recovery
- Survival / Independence
